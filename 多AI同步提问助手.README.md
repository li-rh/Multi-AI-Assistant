# 多AI同步提问助手

> 📌 说明：本脚本参考了 **“多AI同步提问助手”** 与 **“多模型同时回答 & 目录导航”** 两个脚本，在此基础上完成融合与增强。

## ✨ 项目简介

`多AI同步提问助手.js` 是一个 Tampermonkey 用户脚本，用于在多个 AI 页面之间同步提问内容。

你在一个页面发送问题后，脚本会自动将同样内容同步到其他已勾选模型页面，方便并行对比回复效果。

---

## 🧠 主要功能

### 1. 文本同步

- 自动拦截发送请求并提取问题内容
- 支持网络拦截主链路 + 本地事件兜底
- 将文本广播到其他目标标签页并自动发送

### 2. 图片同步（粘贴）

- 支持将粘贴图片同步到已选目标模型页面
- 接收端自动注入上传流程

### 3. 通用文件同步

- 支持三种入口：
  - 粘贴文件（`paste`）
  - 拖拽上传（`drop`）
  - 文件选择（`input[type=file]`）
- 接收端优先使用原生文件输入注入，提升兼容性

### 4. 可视化控制面板

- 右下角悬浮球展开控制面板
- 可勾选目标模型、全选/取消全选
- 可设置常用模型显示
- 徽标显示当前目标数量

### 5. 选择状态同步（可选）

- 开启后：多标签页共享同一套选择状态
- 关闭后：每个标签页独立选择

---

## 🚀 使用方法（重点）

### 快速开始

1. 安装并启用脚本（Tampermonkey）。
2. 打开多个已支持的 AI 页面。
3. 点击右下角悬浮球，勾选要同步的目标模型。
4. 在任一页面正常提问，或粘贴/拖拽/选择文件上传。
5. 其他已选页面会自动同步并发送。

### 推荐使用习惯

- 先打开常用页面，再勾选目标模型。
- 大文件同步会更慢，属于正常现象。
- 若某页面临时不同步，先刷新目标页再重试。

## ⚠️ 注意事项

- 某些站点对“非用户手势上传”限制严格，自动注入可能受浏览器策略影响。
- 建议在排查问题时开启调试日志。
- 仅想单页使用时，请取消其他模型勾选。
